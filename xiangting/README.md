# 享听APP

**技术点：**项目使用MVP模式作架构，通过设计接口实现了逻辑层、界面层和数据层的解耦；使用Glide对图片实现异步加载；使用SQLite保存订阅专辑和使用SharedPreferences实现播放模式的持久化；通过实现自定义FrameLayout实现拉取网络数据时不同状态页面的显示。
